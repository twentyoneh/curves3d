2.
Считайте кривые из файла в контейнер (например, в виде вектора или списка). Путь к файлу должен быть передан с помощью

описана ниже.
3.

4.
Заполните второй контейнер, который будет содержать только круги из первого контейнера. Убедитесь

Отсортируйте второй контейнер в порядке возрастания радиусов окружностей. То есть первый элемент имеет
наименьший радиус, последний - наибольший.
6.
Вычислите общую сумму радиусов всех кривых во втором контейнере.

1.
Поддерживайте несколько типов трехмерных геометрических кривых – окружности, эллипсы и трехмерные спирали. (Упрощенные

аргумент командной строки. Первичное тестирование должно быть выполнено с использованием прикрепленного файла

Выведите идентификатор, название, координаты точек и производных при t=PI/4 для каждой кривой в контейнере.

требования к реализации:
1.

2.

Определения кривых:
-
Все кривые задаются параметрически, т.е. точка вычисляется с использованием некоторой формулы C(t).
-
Окружность параллельна плоскости XoY и определяется ее радиусом и центром.
-
Эллипс параллелен плоскости XoY и определяется двумя его радиусами, расположенными вдоль осей X и Y, и центром.
-
Спираль является пространственной и определяется ее радиусом, шагом (см. рисунок ниже) и центром. Для создания витка требуется 2 * PI в
параметрическом пространстве, т.е. любая точка на спирали удовлетворяет условию C(t + 2*PI) = C(t) +
{0, 0, шаг}.

Метаданные кривой:
-
ID – уникальный 64-разрядный идентификатор
-
Имя – текстовая метка
Структура файла:
-
В первой строке указано количество кривых.
-
Далее приведен список кривых. Каждая строка содержит одну кривую.
-
У каждого типа кривой есть свой собственный тег (дескриптор):
•
“C” для круга
•
“E” для эллипса
•
“H” для спирали
-
Каждая кривая имеет идентификатор и название. Название заключено в кавычки и может содержать только
Латинские символы, цифры и знаки подчеркивания. Имя не содержит пробелов.
-
Каждая кривая содержит определенное местоположение. Местоположение задается трехмерной точкой (три
значения с плавающей запятой после названия).
-
Далее приведены данные, характерные для каждого типа кривой:
•
Радиус окружности – одно значение с плавающей запятой
•
Малый и большой радиусы эллипса – два значения с плавающей запятой
•
Радиус и шаг спирали – два значения с плавающей запятой
Пример:
3
C 11100 “фланец колеса” 0,000000 2,150000 1,650000 6,230000
E 22200 “накладка” 4,550000 0,000000 -1,000000 2,000000 3,000000
H 33300 “bolt_thread” -2,000000 1,000000 5,000000 8,975000 4,256000
Пример содержит 3 кривые:
-
Окружность с радиусом 6,23 и центром в точке (0,00; 2,15; 1,65).
Идентификатор - 11100, имя - wheel_flange.
-
Эллипс с малым радиусом 2,00, большим радиусом 3,00 и центром в точке (4,55; 0,00; -1,00).
Идентификатор - 22200, имя - cover_plate.
-
Спираль с радиусом 8,975, шагом 4,256 и центром в точке (-2,00; 1,00; 5,00).
Идентификатор - 33300, имя - bolt_thread.
